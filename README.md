# Starfall Garden

*Bloom. Blast. Enchant.*

A cozy magical top-down auto-shooter with a living potion-garden to grow. Fly out on moonlit night flights to gather stardust, star hearts, and ingredients, then return to build, upgrade, and enchant your garden. Features night glades, ingredients, wishes, and garden-blight pressure.

## Run

```powershell
npm install
npm run dev
```

## Verify

```powershell
npm run typecheck
npm run build
```

The project is intentionally a single Phaser/Vite/TypeScript game. Scrapline donor files, the optional 3D Scrapyard mode, season pass code, generated build output, and installed dependencies are not kept in the source tree.
