# Neon Scrapline: Factory Raid

Top-down neon extraction auto-shooter with factory progression, raid zones, materials, contracts, and infestation pressure.

## Run

```powershell
npm install
npm run dev
```

## Verify

```powershell
npm run typecheck
npm run build
```

The project is intentionally a single Phaser/Vite/TypeScript game. Scrapline donor files, the optional 3D Scrapyard mode, season pass code, generated build output, and installed dependencies are not kept in the source tree.
